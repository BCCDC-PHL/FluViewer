fluviewer.fluviewer

This module controls the overall flow of analysis by running each analysis stage.

It runs the following analysis stages:

  1. Read Normalization

  2. Assemble Contigs

  3. BLAST Contigs

  4. Scaffolding

  5. Read Mapping

  6. Variant Calling

  7. Consensus Calling

  8. Summary Reporting
